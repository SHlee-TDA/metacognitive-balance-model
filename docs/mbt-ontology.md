## 0. 방법: 타입 체계

Han의 모든 용어를 아래 7개 타입 중 하나(또는 흐려진 조합)로 분류한다.

| 타입 | 정의 | 수학적 대응 |
|---|---|---|
| **상태** (state) | 시간에 따라 값이 변하며 방정식의 미지수가 될 수 있는 것 | 상태변수 x(t) |
| **작용** (operation) | 상태를 바꾸는 사상 — 입력과 변화 대상이 명시되어야 함 | 사상/벡터장 f: 입력 → dx/dt |
| **신호** (signal) | 상태들로부터 읽히는 파생량. 스스로는 아무것도 바꾸지 않고 작용의 입력이 됨 | 관측 함수 y = h(x) |
| **관계** (relation) | 주체와 상태 사이의 2차적 상태 (예: 그 해석에 붙들려 있는가) | 별도 좌표 또는 메타 변수 |
| **능력** (capacity) | 개별 상태가 아니라 **동역학의 성질** — "얼마나 잘 ~할 수 있는가" | 안정성·수렴률 등 계의 성질 |
| **영역** (region) | 상태공간의 부분집합 | Ω ⊂ 상태공간 |
| **실패양상** (failure mode) | 동역학이 보이는 병리적 패턴 | 특정 궤적류/유역 |

핵심 규율: **작용에는 반드시 "무엇을 입력받아 무엇을 바꾸는가"를 묻는다.
상태에는 "무엇이 이것을 바꾸고, 이것은 무엇에 영향을 주는가"를 묻는다.**

## 3. 흐림 지점 목록 (Han이 개념과 작용을 겹쳐 말한 곳)

1. **관점** = "위치"(상태) ⊕ "해석의 방식"(작용) → 해소안: 상태가 작용을 매개변수화.
2. **관점 구성** = 구성하는 과정(작용) ⊕ 구성된 입장(상태). + Han의 P와 사용자의 p의 관계: P = p(형성 가중의 위치) + 내용 방향(L)일 가능성 — **Han에게 확인**.
3. **balance** = 상태 ⊕ 능력 ⊕ 과정. 이론의 1차 정의가 무엇인지 — **Han에게 확인** (능력으로 추정).
4. **CRS** = 조절기(작용)인데 v1은 그 상태(α)에 같은 이름을 썼음 → 표기 분리로 해소 (p 상태 / R 조절기).
5. **m** = "수준"(상태) ⊕ "힘"(능력).
6. **Δ** = 신호인데 하위요소에 '조절 요구'(작용 지시)가 포함 — 신호가 명령을 품음. 조작화 시 분리 필요.
7. **관점 성찰** = '알아차림'(관계 변화) ⊕ 'Δ를 다룸'(조절) — 두 작용이 한 단어.

## 4. 편향 vs 융합 — 수요와 수용성의 분리 (2026-07-15 사용자 논점)

**사용자 판정 채택.** 이전 분석("융합자는 Δ ≈ 0이어야")은 서로 다른 두 개념을 한
슬롯에 뭉갰다:

- **편향**(bias) = p가 simplex의 극단에 있음 — **상태의 성질**. 편향된 사람에게
  성찰의 **객관적 수요**(다른 관점과의 실제 간극)가 가장 큰 것은 수학적으로 당연하고
  개념적으로도 옳다 — convex combination의 극단에서 간극이 최대가 되는 것.
- **융합**(fusion) = 자신의 해석을 관점으로 보지 못함 — **관계**(주체↔p)의 소실.
  "편향되었기 **때문에** 성찰 신호를 가장 수용하지 않을 것" — 수용성은 융합도가 결정.

따라서 지난 설계의 두 공식은 경쟁 관계가 아니라 **서로 다른 슬롯**이었다:

| 슬롯 | 개념 | 타입 | 구체형 |
|---|---|---|---|
| **수요** (demand) | 현재 관점과 대안 사이의 객관적 간극 — "받아야만 하는" 신호 | 신호 | 사용자 공식 계열 (편향 극단에서 최대) — **보류** |
| **수용성** (receptivity/uptake) | 그 신호가 실제로 얼마나 들어오는가 | 관계(융합)의 함수, m이 게이트할 수 있음 | **보류** |

수용된 성찰 신호 = (수요, 수용성)의 결합 — 결합 형태도 보류. **임상적 함의가 구조에서
바로 나온다**: 편향이 클수록 수요는 크고 수용성은 낮다 → 치료 표적이 '내용 교정'이
아니라 '수용성 회복'이어야 하는 이유 (Han 항목 9의 reflective interval과 정확히 합치).
이 수요–수용성 갭이 모델의 심장이 될 가능성이 높다.

## 5. 추상 골격 (타입 시그니처만 — 수식 없음)

```
상태:   L (해석)      p (관점 위치, simplex)      m (관여 수준)      F (융합도/성찰거리)
                                                  [m, F 중 상태로 승격할 것 미결]
신호:   c = conf(L)   Δ = demand(p, 대안들, S)    [구조화: 강도·방향만 조작화]
관계:   F: 주체 ↔ p   (수용성 = g(F, m) — 형태 보류)
작용:   해석   I: (S; p[, P?]) → dL
        조절기 R: (p, 수용된 Δ; u) → dp          ← CRS
        성찰   ℜ: (Δ·수용성) → dF               ← 알아차림 단계
        대안생성 A: 상황 → 후보 관점 집합         ← v1에서 상수로 얼어 있던 작용
        개입   u: 외생 — (F, p)에 작용            ← reflective interval
능력:   balance-capacity, tolerance — 위 동역학이 정의된 뒤 그 성질로 사후 정의
영역:   adaptive zone = 능력이 성립하는 상태공간 부분집합
```

결합계의 변수 수(3개인지 4개인지: L, p 외에 m·F의 지위)는 개념 분석의 **출력**으로
결정한다 — 아직 미확정.
