## 0. 용어 정돈: b를 무엇이라 부를 것인가

**문제.** v1은 b를 belief(믿음)라 불렀으나, 심리학에서 belief는 신념 체계·망상적
신념 등 전혀 다른 함의를 갖는다 (Han의 읽기 4분해에서도 "belief content"는 **어떤
해석이 떠올랐는가** = 우리의 ĥ에 해당하지, 등급화된 확률 b가 아니다).
confidence로 바꾸면 통계학의 confidence(신뢰구간)와 노트 안의 c(t) 정의와 충돌.

**제안: b = credence (크리던스, 국문 병기: 주관적 확률).**

- **형식인식론(formal epistemology)의 표준 용어**로, 정확히 "등급화된 믿음의 정도
  = 주관적 확률"을 뜻한다. 이 분야가 이 단어를 만든 이유 자체가 범주적 belief와
  등급적 degree-of-belief를 구분하기 위해서였다 — 우리의 필요와 동일.
- 임상 심리학에서 무부하(신념·확신·병식 어느 것과도 충돌하지 않음), 통계학에서도
  베이지언 주관 확률로 자연스럽게 읽힘.
- 결과적으로 세 층이 깔끔히 분리된다: **ĥ = 해석 내용**(Han의 belief content),
  **b = 그 해석에 대한 credence**(등급적 확률), **c = confidence**(그 credence에서
  파생되는 메타인지적 느낌, Proust 앵커 유지).

차선 후보(기각 사유): plausibility(Jaynes 계열이나 상태변수 명칭으로 비표준),
endorsement(망상 연구 용어로 부분 부하), subjective probability(정확하나 장황).

## 1. v2의 구조: 2-결합계 → 3-결합계

사용자 방향(2026-07-15): 관점 P를 창발적 readout이 아니라 **세 번째 동역학 변수**로
승격. Han의 사슬 [관점 성찰 → Δ 인식 → Δ 조절 → P 재구성]과 구조도
(Self+Other/Context → Δ → adaptive zone 조절 → P → Self·Other/Context로 **feedback**)를
동역학으로 구현하는 것.

**상태변수 3개:**

| 변수 | 의미 | 시간 척도 |
|---|---|---|
| L(t) ∈ ℝ | 해석 상태 (credence b = σ(L)) | 빠름 (순간의 해석) |
| **a**(t) = (a_self, a_context) ∈ ℝ²₊ | 참조 가중 벡터 — 방향 α̂ = 배분, 크기 m = 전체 관여 | 중간 (세션 내 조절) |
| P(t) | 관점 구성 — 해석적 입장 | 느림 (입장의 고착·재구성) |

**자극**: S(t) = (s_self(t), s_outer(t)) — 내부 재료와 외부 재료의 두 성분.
v1과의 연속성을 위해 1차 후보는 s_self := λ_S(L, P), s_outer := λ_C(t, P)
(가공된 채널 신호; 원자극을 쓸지 채널 출력을 쓸지는 미결 선택 D4).

**결합 구조** (사용자 명세 + Han 구조도):

```
                    ┌──────────── P가 채널을 편향 (Han: P → Self/Context feedback) ───────────┐
                    ▼                                                                          │
  S(t) ──→ [가중: a] ──→ dL/dt  (해석)                                                         │
                    │                                                                          │
                    ├──→ Δ(t) = f(S, a)  (차이의 지각)                                          │
                    │         │                                                                │
                    │         └──→ 관점 성찰 R(Δ, ...) ──→ dP/dt  (관점 재구성) ────────────────┘
                    │                                    │
  u(t) ──────────→ da/dt = G(a, L, L̇, P; u)  ←───────────┘  (관점이 가중 조절의 입력)
```

## 2. Δ의 조작화 — 사용자 공식과 모서리 사례 분석 ★논의 필요

**사용자 제안**: Δ(t) = |a_self(t)·s_self(t) − a_context(t)·s_outer(t)|.

이 공식은 "**가중된 두 밀기의 불균형**"(net drive imbalance)이다. 훌륭한 성질:
가중치가 들어가 있어 **같은 자극도 사람·순간마다 다르게 느껴진다**는 요구를 충족하고,
Δ가 (S, a)의 함수이므로 관점 성찰도 자동으로 (S, a)의 함수가 된다.

**모서리 사례 점검** (s_self = +2 위협 찌름, s_outer = −1 안심 단서로 고정):

| 상태 | **a** | 사용자 공식 Δ | 임상적으로 기대되는 "felt Δ" |
|---|---|---|---|
| 균형·고관여 | (0.8, 0.8) | \|1.6+0.8\| = 2.4 | 큼 ✓ (갈등을 뚜렷이 느낌) |
| 균형·저관여 (20,20) | (0.2, 0.2) | 0.6 | 작음 ✓ (Han: "알아차리는 힘이 약함") |
| **자기 융합** | (1.6, 0) | \|3.2\| = **3.2 (최대)** | **≈ 0이어야** — 융합 = 다른 관점이 안 보임 = 간극을 못 느낌 |

마지막 행이 문제다: 완전히 한 관점에 융합된 사람이 공식상 가장 큰 Δ를 느끼게 된다.
그런데 Han의 정의에서 Δ는 "현재 지배적 관점과 **다른 가능한 관점들** 사이의 차이"이고,
융합의 병리는 바로 그 차이가 **느껴지지 않는 것**이다 (관점 성찰 = 낮은 문턱에서
그것을 다시 느끼게 하는 활동).

**정제 제안 — 두 신호로 분리:**

1. **δ(t) = a_self·s_self − a_context·s_outer** (부호 있는 **구동 불균형**) — 사용자
   공식의 부호 보존판. 계가 어느 쪽으로 끌려가는지(불일치 방향, Han의 하위요소)를
   나타내며, **pivot의 발동 신호**로 적합 (v1의 b·L̇ 트리거를 일반화).
2. **Δ̂(t) = (a_self·a_context)·|s_self − s_outer|** (**지각된 관점 간 차이**) — 두
   관점을 **동시에 붙들고 있어야만** 그 차이가 느껴진다는 성질을 곱 a_self·a_context가
   구현. 검산: 자기 융합 (1.6, 0) → Δ̂ = 0 ✓; (20,20) → 0.04·3 = 0.12 (약함) ✓;
   (50,50) → 0.25·3 = 0.75 (중간) ✓; (80,80) → 0.64·3 = 1.92 (강함) ✓ — **Han의 세
   사례가 자동 재현**되고, m 게이팅(1차 초안의 A안)이 별도 함수 없이 내장된다.
   관점 성찰 R의 구동 신호로 적합.

두 신호 모두 "Δ의 한 조작화"(Han 지침)로 명시. — **결정 D1**: 사용자 공식(δ)만 쓸지,
δ + Δ̂ 이원 신호로 갈지.

## 3. 세 방정식의 1차 후보형

$$\textbf{(해석)}\quad \frac{dL}{dt} = a_{\text{self}}\,\lambda_S(L, P) + a_{\text{context}}\,\lambda_C(t, P) - \kappa L$$

- λ_S(L, P) = a_S + β_S·P + r·σ(L) — 관점이 내부 읽기를 편향 (피해적 관점 P>0이면
  같은 내부 재료도 위협 쪽으로 읽힘)
- λ_C(t, P) = a_C + β_C·P + ξ(t) — **관점이 외부 읽기도 편향** — P>0이면 외부 단서마저
  위협 확증적으로 읽힘 → **context-dominant JTC(B-03)와 감사 F3(외부 채널 위협 확증,
  B-05)가 이 한 항에서 함께 해결됨**. Han 구조도의 "P → Self·Other/Context feedback"의 구현.

$$\textbf{(가중 조절)}\quad \frac{d\mathbf{a}}{dt} = \mathcal{G}(\mathbf{a}, L, \dot L, P;\, u) = -\Gamma\,(\mathbf{a} - \mathbf{a}_0(P)) + \mathbf{u}(t) + \mathbf{R}_a(\hat\Delta, \delta)$$

- 사용자 명세대로 P가 G의 입력: 고착된 관점은 가중의 평형점 자체를 끌어당김
  (**a**₀(P): 피해적 관점 → 자기 쪽 가중의 set-point 상승 — "관점이 반추를 부른다")
- **R_a** = 관점 성찰의 가중 조절 성분: Δ̂를 지각했을 때 부호 δ의 반대쪽으로 재배분
  ("내 관점을 어디에 비중을 두도록 할지 변화시키는 작용" — 사용자 정의)
- u(t) = 임상 개입 (pivot v2: 방향 전환 + reflective interval — κ 일시 증가·커밋 임계 일시 상승)

$$\textbf{(관점 재구성)}\quad \frac{dP}{dt} = \mathcal{R}(\hat\Delta, L, \hat\alpha, P) = -\gamma_P\,\big(P - \phi(L, \hat\alpha)\big)\cdot w(\hat\Delta) + (\text{고착 항})$$

- 기본 성질: P는 **느린 변수** — 빠른 계 (L, α̂)가 머무는 자리를 향해 서서히 이동
  (입장의 습관화). Δ̂가 지각되고 견뎌질 때(w(Δ̂) 큼) 재구성이 **빨라짐** — Han의
  "Δ는 관점 성찰과 조절이 시작되는 지점"의 구현.
- **부수 수확**: 이 느린 층은 챕터가 요구한 between-session 재보정(§4.2 temporal
  multilayering, 감사 F4/백로그 B-06)의 자리이기도 하다 — 3-결합계가 시간 다층성을
  공짜로 얻는다.

## 4. 읽기(readout) — Han의 4분해와의 대응

| Han | v2 | 비고 |
|---|---|---|
| belief content | ĥ = argmax (해석 내용) | 용어 정돈: b는 credence로 개명 |
| confidence | c = \|2b − 1\| | 유지 |
| insight / reflective distance | Δ̂ 지각 능력 자체, 또는 1 − fusion(P에의 고착도) | P가 상태변수가 되어 fusion을 P 동역학에서 직접 정의 가능 |
| action tendency / safety distance | 행동 게이트 (B-20): act iff c > θ_act ∧ 간격 조건 | 신규 |
